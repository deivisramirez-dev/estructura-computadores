Introducción a Almacenamiento e I/O

Sistemas de Almacenamiento

Los sistemas de almacenamiento son componentes fundamentales que permiten el almacenamiento persistente de datos y programas. Incluyen desde dispositivos de almacenamiento primario hasta sistemas de almacenamiento secundario y terciario.

  • Almacenamiento Primario: RAM, ROM, caché
  • Almacenamiento Secundario: Discos duros, SSDs
  • Almacenamiento Terciario: Cintas, almacenamiento en la nube

Dispositivos de Entrada/Salida

Los dispositivos de entrada/salida permiten la comunicación entre el computador y el mundo exterior. Estos dispositivos varían en velocidad, capacidad y propósito.

  • Dispositivos de Entrada: Teclado, ratón, escáner
  • Dispositivos de Salida: Monitor, impresora, altavoces
  • Dispositivos de Almacenamiento: Discos, memorias USB

Tipos de Almacenamiento

Almacenamiento Volátil

  • RAM: Memoria de acceso aleatorio
  • DRAM: RAM dinámica (requiere refresco)
  • SRAM: RAM estática (más rápida)
  • Características: Alta velocidad, pérdida de datos al apagar

Almacenamiento No Volátil

  • ROM: Memoria de solo lectura
  • Flash: Memoria flash (SSDs, USB)
  • HDD: Disco duro magnético
  • Características: Persistencia de datos, menor velocidad

Dispositivos de Entrada/Salida

Categorías de Dispositivos

Dispositivos de Entrada

  • Teclado: Entrada de texto y comandos
  • Ratón: Control de puntero
  • Escáner: Digitalización de documentos
  • Micrófono: Entrada de audio
  • Cámara: Entrada de video

Dispositivos de Salida

  • Monitor: Visualización de información
  • Impresora: Salida en papel
  • Altavoces: Salida de audio
  • Proyector: Visualización ampliada
  • Actuadores: Control de dispositivos físicos

Dispositivos de Almacenamiento

  • Disco Duro: Almacenamiento magnético
  • SSD: Almacenamiento flash
  • Memoria USB: Almacenamiento portátil
  • Tarjeta SD: Almacenamiento para dispositivos móviles
  • Cinta: Almacenamiento de respaldo

Modos de Transferencia

Polling

La CPU verifica constantemente el estado del dispositivo

Ventajas:
  • Simple de implementar
  • Control directo
Desventajas:
  • Ineficiente
  • Consume CPU

Interrupciones

El dispositivo notifica a la CPU cuando está listo

Ventajas:
  • Eficiente
  • Respuesta inmediata
Desventajas:
  • Complejo
  • Overhead de contexto

DMA

Transferencia directa entre memoria y dispositivo

Ventajas:
  • Máxima eficiencia
  • CPU libre
Desventajas:
  • Complejo
  • Costoso

Tipos de Buses

Buses Paralelos

  • Características: Múltiples líneas de datos
  • Ventajas: Alta velocidad, transferencia simultánea
  • Desventajas: Interferencia, costoso
  • Ejemplos: PCI, ISA

Buses Seriales

  • Características: Una línea de datos
  • Ventajas: Menos interferencia, económico
  • Desventajas: Menor velocidad
  • Ejemplos: USB, SATA, PCIe

Buses Síncronos

  • Características: Reloj común
  • Ventajas: Sincronización precisa
  • Desventajas: Limitaciones de frecuencia
  • Ejemplos: PCI, DDR

Buses Asíncronos

  • Características: Sin reloj común
  • Ventajas: Flexibilidad, escalabilidad
  • Desventajas: Complejidad
  • Ejemplos: USB, Ethernet

Funciones de los Controladores

Traduce entre el protocolo del bus y el protocolo del dispositivo

Almacena temporalmente datos para compensar diferencias de velocidad

Gestiona las operaciones específicas del dispositivo

Detecta y maneja errores de comunicación y dispositivo

Tipos de Controladores

Controladores de Disco

  • IDE: Interfaz de disco integrada
  • SATA: Serial ATA
  • SCSI: Small Computer System Interface
  • NVMe: Non-Volatile Memory Express

Controladores de Red

  • Ethernet: Red local
  • WiFi: Red inalámbrica
  • Bluetooth: Red personal
  • Fibra: Red de alta velocidad

Controladores de Video

  • VGA: Video Graphics Array
  • DVI: Digital Visual Interface
  • HDMI: High Definition Multimedia Interface
  • DisplayPort: Interfaz de pantalla digital

Protocolos de Comunicación

Protocolos Comunes

USB (Universal Serial Bus)

Versiones: USB 1.1, 2.0, 3.0, 3.1, 4.0

Velocidad: 12 Mbps - 40 Gbps

Características: Plug and Play, alimentación

Aplicaciones: Teclados, ratones, almacenamiento

SATA (Serial ATA)

Versiones: SATA I, II, III

Velocidad: 1.5 - 6 Gbps

Características: Conexión punto a punto

Aplicaciones: Discos duros, SSDs

PCIe (PCI Express)

Versiones: PCIe 1.0 - 6.0

Velocidad: 250 MB/s - 64 GB/s

Características: Arquitectura punto a punto

Aplicaciones: Tarjetas gráficas, SSDs NVMe

WiFi (IEEE 802.11)

Estándares: 802.11a/b/g/n/ac/ax

Velocidad: 11 Mbps - 9.6 Gbps

Características: Comunicación inalámbrica

Aplicaciones: Redes locales, Internet
