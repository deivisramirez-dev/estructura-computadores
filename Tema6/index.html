Arquitectura de Buses

Sistemas de buses y transferencia de datos

Tipos de Buses

Bus del Sistema

Conecta CPU, memoria y controladores principales

  • Alta velocidad de transferencia
  • Baja latencia
  • Protocolos optimizados
  • Arbitraje centralizado

Bus de Expansión

Conecta tarjetas de expansión y periféricos

  • PCI Express
  • USB
  • SATA
  • Thunderbolt

Bus de Memoria

Conecta CPU con memoria principal

  • DDR4/DDR5
  • Alto ancho de banda
  • Baja latencia
  • Protocolos especializados

Características de los Buses

Ancho de Banda

Capacidad de transferencia de datos

  • Medido en bits por segundo
  • Determina la velocidad máxima
  • Depende del ancho del bus

Frecuencia

Velocidad de operación del bus

  • Medida en Hz
  • Influye en la latencia
  • Determina el rendimiento

Arbitraje

Mecanismo de control de acceso

  • Prioridad de dispositivos
  • Prevención de conflictos
  • Distribución del ancho de banda

Acceso Directo a Memoria

Transferencia de datos sin intervención de la CPU

Concepto de DMA

El DMA (Direct Memory Access) permite que los dispositivos periféricos accedan directamente a la memoria principal sin intervención de la CPU, mejorando significativamente el rendimiento del sistema.

Ventajas del DMA

  • Mayor rendimiento: La CPU puede realizar otras tareas
  • Menor latencia: Acceso directo sin overhead
  • Eficiencia energética: Menor uso de CPU
  • Escalabilidad: Múltiples transferencias simultáneas

Operación del DMA

Solicitud

El dispositivo solicita transferencia DMA

Configuración

Se configuran direcciones y tamaño

Transferencia

Se realiza la transferencia de datos

Finalización

Se notifica la finalización a la CPU

Protocolos Principales

USB (Universal Serial Bus)

Estándar para conexión de dispositivos

  • USB 2.0: 480 Mbps
  • USB 3.0: 5 Gbps
  • USB 3.1: 10 Gbps
  • USB 4.0: 40 Gbps

PCI Express

Bus de expansión de alta velocidad

  • PCIe 3.0: 8 GT/s
  • PCIe 4.0: 16 GT/s
  • PCIe 5.0: 32 GT/s
  • Lanes configurables

SATA (Serial ATA)

Interfaz para dispositivos de almacenamiento

  • SATA I: 1.5 Gbps
  • SATA II: 3 Gbps
  • SATA III: 6 Gbps
  • Hot-plugging
