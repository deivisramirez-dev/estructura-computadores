Organización de los diferentes niveles de memoria

Niveles de Memoria

Registros

Más rápidos, menor capacidad, mayor coste

  • Velocidad: ~1ns
  • Capacidad: 32-64 bytes
  • Coste: Muy alto
  • Ubicación: CPU

Cache L1

Memoria cache de primer nivel

  • Velocidad: ~2-4ns
  • Capacidad: 32-64 KB
  • Coste: Alto
  • Ubicación: CPU

Cache L2

Memoria cache de segundo nivel

  • Velocidad: ~10ns
  • Capacidad: 256 KB - 1 MB
  • Coste: Medio
  • Ubicación: CPU o chipset

RAM

Memoria principal del sistema

  • Velocidad: ~50-100ns
  • Capacidad: 8-64 GB
  • Coste: Bajo
  • Ubicación: Placa base

Almacenamiento

Dispositivos de almacenamiento masivo

  • Velocidad: ~5-15ms
  • Capacidad: 1-10 TB
  • Coste: Muy bajo
  • Ubicación: Interno/Externo

Sistemas de Cache

Memoria cache y técnicas de optimización

Tipos de Cache

Cache de Instrucciones

Almacena instrucciones del programa

  • Reduce accesos a memoria principal
  • Mejora el rendimiento del pipeline
  • Optimiza la localidad temporal

Cache de Datos

Almacena datos del programa

  • Acceso rápido a variables
  • Reduce latencia de memoria
  • Optimiza la localidad espacial

Cache Unificado

Almacena tanto instrucciones como datos

  • Flexibilidad en la asignación
  • Mejor utilización del espacio
  • Complejidad de gestión

Políticas de Cache

Política de Mapeo

  • Directo: Una ubicación fija
  • Asociativo: Múltiples ubicaciones
  • Set-Asociativo: Compromiso entre ambos

Política de Escritura

  • Write-Through: Escribe en cache y memoria
  • Write-Back: Escribe solo en cache
  • Write-Buffer: Buffer intermedio

Política de Reemplazo

  • LRU: Menos recientemente usado
  • FIFO: Primero en entrar, primero en salir
  • Aleatorio: Selección aleatoria

Memoria Virtual

Sistema de gestión de memoria virtual

Conceptos de Memoria Virtual

Paginación

División de memoria en páginas de tamaño fijo

  • Páginas de 4KB típicas
  • Tabla de páginas para mapeo
  • TLB para acelerar traducciones
  • Manejo de fallos de página

TLB (Translation Lookaside Buffer)

Cache para traducciones de direcciones virtuales

  • Acceso rápido a traducciones
  • Reducción de accesos a tabla de páginas
  • Fallos de TLB manejados por hardware
  • Reemplazo de entradas TLB

Protección de Memoria

Mecanismos de seguridad y aislamiento

  • Separación usuario/supervisor
  • Permisos de lectura/escritura/ejecución
  • Aislamiento entre procesos
  • Manejo de violaciones de acceso

Almacenamiento Masivo

Tecnologías de almacenamiento permanente

Tecnologías de Almacenamiento

Discos Duros (HDD)

Almacenamiento magnético tradicional

  • Platos magnéticos giratorios
  • Cabezales de lectura/escritura
  • Capacidad: 1-20 TB
  • Velocidad: 100-200 MB/s

SSDs (Solid State Drives)

Almacenamiento basado en memoria flash

  • Sin partes móviles
  • Mayor velocidad de acceso
  • Capacidad: 250GB - 8TB
  • Velocidad: 500MB/s - 7GB/s

NVMe (Non-Volatile Memory Express)

Interfaz de alta velocidad para SSDs

  • Protocolo PCIe optimizado
  • Baja latencia
  • Alto throughput
  • Escalabilidad

Nuevas tecnologías de memoria y almacenamiento

Tecnologías del Futuro

Memoria Resistiva (ReRAM)

Memoria no volátil basada en resistencia

  • Alta densidad
  • Bajo consumo energético
  • Escalabilidad
  • Aplicaciones en IA

Memoria Magnética (MRAM)

Memoria no volátil basada en magnetismo

  • Velocidad de SRAM
  • No volatilidad
  • Resistencia a radiación
  • Aplicaciones espaciales

Almacenamiento en ADN

Almacenamiento molecular en ADN sintético

  • Densidad extrema
  • Durabilidad milenaria
  • Bajo mantenimiento
  • Aplicaciones de archivo
